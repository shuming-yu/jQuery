jQuery.extend()

  • 功能： 可以合併兩個或更多個物件的內容到第一個 target 參數的物件之中。
  • 改變： 改變第一個參數的物件。
  • 語法： jQuery.extend( target, object1 [, objectN ] )。
  • 語法： jQuery.extend( [deep ], target, object1 [, objectN ] )。
  • 語法： jQuery.extend( object )。
  • 回傳值： Object。

jQuery.fn.extend()

  • 功能： 將一個物件的內容合併到 jQuery prototype 上面，以在 jQuery 實例上提供新的方法。
  • 語法： jQuery.fn.extend( object )。
