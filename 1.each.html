jQuery.each()

  • 功能： 對陣列的每個元素執行一次提供的函數。
  • 改變： 不會直接改變原陣列，但可能會依帶入的函式而改變。可使用 return 結束迴圈。
  • 語法： jQuery.each( array, callback )。
  • 語法： jQuery.each( object, callback )。
  • 回傳值： jQuery。
  • 參數： ( array || object ) 與 callback 函式

.each()

  • 語法： $( selector ).each( function )。
