jQuery.inArray()

  • 功能： 函數用於在陣列中查找指定值，並返回它的索引值index （ 如果未滿足條件，則回傳 -1 ）。
  • 語法： jQuery.inArray( value, array [, fromIndex ] )。
  • 回傳值： Number。
  • 參數： value 與 array 與 fromIndex ( Type: Number 指定索引位置開始查找, 預設值為 0, 第三參數可忽略 )
